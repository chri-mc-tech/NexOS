# NexOS

NexOS è un sistema operativo sperimentale che sto sviluppando per imparare il funzionamento di un OS a basso livello

È un progetto open source personale

---

## Stato attuale

- si avvia con un bootloader BIOS in assembly
- carica il kernel in memoria
- il kernel stampa un output base

---

## Tecnologie

- Assembly x86 (bootloader BIOS)
- C++ (kernel)

---

## Obiettivo

Voglio capire e costruire un sistema operativo passo dopo passo, partendo dalle basi

---

## Versioni

MAJOR.MINOR.PATCH

- MAJOR: cambiamenti grandi
- MINOR: nuove funzionalità
- PATCH: fix e miglioramenti
